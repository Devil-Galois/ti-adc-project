# ADC 规格书

## 1. 总体规格

| 项目 | 规格 / 状态 |
|---|---|
| 应用场景 | 56 Gb/s PAM-4 接收机 |
| 工艺 | 12 nm FinFET |
| 架构 | 32 通道时间交织 SAR ADC |
| 交织层级 | `2 x 2 x 8` |
| 总采样率 | 28 GS/s |
| 子 ADC 采样率 | 875 MS/s |
| 量化位数 | 7-bit + 1-bit redundancy |
| 输入共模 | 550 mV |
| 差分输入摆幅 | 600 mVpp |
| 输入信号 | 差分正弦测试，系统级 FFT 点数 1024 |
| 校准对象 | 失调、增益、采样时序偏斜 |
| 当前验证阶段 | 原理图级前仿真 + MATLAB/Verilog 校准验证 |

## 2. 性能指标

| 项目 | 目标 | 前仿真结果 / 说明 |
|---|---:|---|
| ENOB | >= 5.5-bit | TT 近 Nyquist 输入时 > 6-bit |
| SFDR | >= 45 dBc | TT 近 Nyquist 输入时 > 46 dBc；代表值 49.3 dBc |
| SNDR | - | 代表值 35.6 dB |
| Core Power | - | 88.31 mW，不含片外 DSP |
| Walden FoM | - | 48.6 fJ/conv-step |
| 输入缓冲器 SFDR | 前端不限制系统 SFDR | 目标输入频率范围内 > 48 dB |

## 3. 架构规格

| 模块 | 规格 |
|---|---|
| 顶层结构 | 阻抗匹配网络、分层交织采样前端、32 个子 SAR ADC、多相时钟/DCDL、片外 DSP |
| 第一级交织 | 2 路输入缓冲器，每个驱动 2 个 Rank-1 自举采样器 |
| 第二级交织 | 4 路第一级采样保持输出，每路经子缓冲器驱动 8 个子 SAR ADC |
| 时序敏感相位 | 第二级结构将需要 DCDL 校准的主相位约束为 4 相 |
| 数据输出 | 32 路子 ADC 数据重组后进行系统级动态性能评估 |

## 4. 模块规格

| 模块 | 设计规格 / 约束 |
|---|---|
| 输入缓冲器 | 源跟随器，1.2 V 供电，后级等效负载为最差 `45 ohm` 导通电阻和 `80 fF` 采样电容；不同 corner 下带宽大于 22 GHz |
| Rank-1 采样器 | 全分裂自举开关；主采样窗口约 70 ps；自举电压提升约 130 mV，导通提前超过 6.5 ps |
| 子缓冲器 | 全差分 Class-AB 超级源跟随器；1.2 V 供电；后级等效 `40 ohm + 120 fF` 负载；Nyquist 附近输出阻抗小于 40 ohm；所有 corner 下最差阶跃建立时间小于 80 ps |
| Rank-2 采样器 | 时钟升压采样开关；1 V 供电，升压电容 20 fF；不同 corner 下开关导通电阻不超过 40 ohm |
| CDAC | 恒定共模、分裂电容、单调开关切换；4C 后重复插入 4C 实现 1-bit redundancy；单位电容 0.5 fF；含寄生后采样总电容约 120 fF |
| 基准缓冲器 | 基于 SSF 的基准缓冲器；基准电压范围 550 mV 至 650 mV；片上去耦电容 4.5 pF；配合 redundancy 覆盖最坏转换建立误差 |
| 比较器 | Strong ARM 动态比较器；比较器及后级逻辑 0.8 V 供电；输入共模选取 500 mV；TT 下噪声 1.22 mV、判决时间 18.5 ps、能耗 20.4 fJ |
| SAR 逻辑 | TSPC 动态触发器链与异步时钟；最慢 corner 下单次转换仍预留 84 ps 以上时序裕量 |
| 多相时钟 | 14 GHz 差分时钟经 CML 二分频得到 4 相 7 GHz 时钟，再经分频、移位寄存器和占空比调节得到 32 相 875 MHz 子采样时钟 |
| DCDL | 可调负载电容型数控延迟链；单边覆盖约 3.81 ps；最差步进约 87 fs；用于后台 Skew 闭环调节 |

## 5. 失配预算与校准接口

| 失配来源 | 估计范围 / 设计含义 |
|---|---|
| 采样前端失调 | 99.7% 分位约 14 mV |
| 采样前端增益 | 99.7% 分位约 0.4% |
| 基准电压失配 | 对增益失配贡献约 8% |
| CDAC 总电容失配 | 对增益失配贡献约 5% |
| 比较器失调 | `3 sigma` 约 18 mV |
| 时序偏斜 | DCDL 需覆盖约 `-3.4 ps` 至 `+3.4 ps` 的主相位残余 skew |

校准实现位于 `dsp/`，本规格书只描述接口关系：

- OS/Gain 校准：ADC0 作为参考通道，片外 DSP 使用 EMA 提取均值和绝对值均值，再通过 LMS 更新失调和增益校准寄存器。
- Skew 校准：DSP 对经 OS/Gain 校准后的 32 通道数据按 4 相重组，使用 MAD 统计量检测偏斜方向，输出 3-bit 符号码。
- DCDL 控制：Skew 符号码经 SPI 写入 ADC 片上寄存器，控制 DCDL 码按 1 LSB 递增或递减，形成后台闭环。
